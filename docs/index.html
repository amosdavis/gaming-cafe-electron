📋 Installation Instructions

  1. Download the ISO above and write it to a USB drive using Rufus (GPT + UEFI mode) or burn it to a DVD.
  2. Set the target PC to boot from USB/DVD — enter BIOS/UEFI (F2 / Del) and set USB as the first boot device, or press F12 for the boot menu.
  3. Boot and wait — Windows Setup runs fully automatically. No prompts. The install takes approximately 15–25 minutes depending on hardware.
  4. First reboot — the kiosk setup script runs, installs the Gaming Cafe Kiosk app, hardens the OS, and sets the kiosk as the Windows shell.
  5. Done — the PC will reboot into the kiosk. Log in to the admin panel to add users and credits.

🌐 PXE Quick-Start

  1. Download PXE Boot Files and extract pxe-files.zip.
  2. Copy boot.wim to your WDS server or TFTP root.
  3. Add boot.wim as a Boot Image in WDS or serve via wimboot + iPXE using the included pxe-ipxe.cfg.
  4. Boot kiosk PCs from the network — installation proceeds automatically.
